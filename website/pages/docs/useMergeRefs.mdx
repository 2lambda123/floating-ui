# useMergeRefs

<Notice type="version" gap="above">
  Since `v0.15.0`
</Notice>

Merges an array of refs, returning a memoized callback ref or
`null{:js}`.

```js
import {useMergeRefs} from '@floating-ui/react';
```

## Usage

This hook is a utility hook.

```js
function App() {
  const ref = useMergeRefs([ref1, ref2, ref3]);
  return <div ref={ref} />;
}
```
